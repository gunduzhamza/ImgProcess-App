# Image Processing Applications
#### This project contains examples of image processing applications. I made an user interface by using pyqt5 libraries.

![pic](./Readme-img/window.png)

- #### HW-1
  - Turn right
  - Turn left

  - Negative

  - Log Transformation
- #### HW-2

  - Histogram Graph
  - Histogram Equalization
- #### HW-3
  - Derivative
  - Low Pass Filter
  - High Pass Filter
- #### HW-4
  - Otsu Threshold
  - Erosion
  - Dilation
  - Opening
  - Closing
  - Boundry Extraction
